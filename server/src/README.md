# Server source structure

This folder is organized into standard layers for a Node.js service:

- `config/`: environment configuration and setup.
- `controllers/`: request handlers and orchestration.
- `middlewares/`: shared request/response middleware.
- `models/`: data models and persistence helpers.
- `routes/`: route definitions and versioned routers.
- `services/`: business logic and integrations.
- `utils/`: shared helpers and utilities.
