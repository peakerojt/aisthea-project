# Glossary tieng Viet cho AISTHEA

## Thuat ngu chinh

- `Product` -> `San pham`
- `Category` -> `Danh muc`
- `Collection` -> `Bo suu tap`
- `Variant` -> `Bien the`
- `Inventory` / `Stock` -> `Ton kho`
- `In stock` -> `Con hang`
- `Out of stock` -> `Het hang`
- `Low stock` -> `Sap het hang`
- `Cart` / `Bag` -> `Gio hang`
- `Checkout` -> `Thanh toan`
- `Order` -> `Don hang`
- `Return` -> `Tra hang`
- `Refund` -> `Hoan tien`
- `Customer` -> `Khach hang`
- `Save` -> `Luu`
- `Cancel` -> `Huy`
- `Delete` -> `Xoa`

## Trang thai don hang

- `PENDING` -> `Cho xac nhan`
- `PROCESSING` -> `Dang xu ly`
- `SHIPPING` / `SHIPPED` -> `Dang giao hang`
- `DELIVERED` -> `Da giao hang`
- `COMPLETED` -> `Hoan thanh`
- `CANCELLED` -> `Da huy`
- `RETURN_REQUESTED` -> `Yeu cau tra hang`
- `RETURNED` -> `Da tra hang`
- `REFUNDED` -> `Da hoan tien`

## Quy uoc bo sung

- Storefront uu tien cau ngan, mem va de mua.
- Admin uu tien thao tac ro rang, dung nghiep vu.
- Toast va feedback uu tien dong tu ngan gon, tranh dau cham than khong can thiet.
