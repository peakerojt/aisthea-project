# Locale VI Refined Package

Gói này chứa bản refactor tiếng Việt chuẩn ecommerce cho toàn bộ locale JSON hiện có.

## Nguyên tắc đã áp dụng
- Ưu tiên tiếng Việt tự nhiên cho storefront
- Đồng nhất thuật ngữ admin
- Chuẩn hóa trạng thái đơn hàng/hoàn trả
- Giảm copy literal từ tiếng Anh
- Rút gọn toast, CTA, placeholder
