# Style guide locale VI ecommerce

## 1. Tone theo man hinh

- Storefront: ngan, tu nhien, de mua, tranh ky thuat.
- Admin: ro thao tac, dung nghiep vu, nhat quan.
- Fashion / brand: mem, co chat thoi trang, tranh van dich may.

## 2. Quy tac viet copy

- CTA bat dau bang dong tu ro nghia.
- Trang thai ngan, dung mot cach goi tren toan he thong.
- Placeholder uu tien cach nguoi dung Viet thuc su nhap lieu.
- Toast ngan, co the bo dau cau neu thong diep da du ro.

## 3. Mau uu tien

- `Them vao gio hang`
- `Thanh toan ngay`
- `Luu thay doi`
- `Cap nhat trang thai`
- `Yeu cau tra hang`

## 4. Khong nen

- Dung cung mot khai niem voi nhieu cach dich.
- Dung cau qua sat nghia tieng Anh.
- Tron tone admin vao copy ban hang.
