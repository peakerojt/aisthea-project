# PR checklist cho locale VI

- Da dung dung glossary chung cua du an.
- CTA bat dau bang dong tu ro nghia va khong qua dai.
- Trang thai khong bi lech voi `orders`, `enums`, `analytics` va config hard-code.
- Copy storefront nghe tu nhien hon copy admin.
- Placeholder va label hop voi cach nguoi dung Viet nhap lieu.
- Toast va feedback ngan, khong dung qua nhieu dau cham than.
- Khong doi key locale neu khong that su can thiet.
- Kiem tra interpolation nhu `{{count}}`, `{{amount}}`, `{{page}}`.
- Chay `npm run i18n:check` truoc khi merge.
